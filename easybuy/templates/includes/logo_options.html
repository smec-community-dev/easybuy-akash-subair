How to Use

  1. Choose your favorite logo design from above
  2. Copy the HTML code for that option
  3. Replace the current logo in core_base.html
  4. The logo is already responsive and works in dark mode

Recommendation: Option 1 (current), Option 9 (gradient text), or Option 11 (dual color) work best for e-commerce.

All logos use Material Symbols icons and Tailwind CSS - no external images needed!
